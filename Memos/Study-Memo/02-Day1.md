# 第一周学习小结

## Git

1. 本质：一套**协议**，实现了分布式版本管理
2. 项目操作步骤
- Repo Dir, 建仓库
- Add, 加文件
- Commit, 提交创建或修改Commit Change
- Push Remote, 各成员反馈意见
- Merge, 编辑汇总、合并等
- Release, 合并发行
3. 远程协作：
+ fork
+ pull request

## markdown
1. 一种轻量级标记语言,当前许多网站都广泛使用 Markdown 来撰写帮助文档或是用于论坛上发表消息。例如：GitHub、reddit等
